## ФАЗА 0 — Разведка и доступы (без кода, без денег)

Цель: понять, возможен ли «возврат» или это «наследник». Всё зависит от основателя.

- [ ] 🟡 Поговорить с основателем (см. `07_FOUNDER_PITCH.md`)
- [ ] 🔴 Получить **дамп MongoDB** (прогресс игроков, донаты, кланы) — без него нет возврата аудитории
- [ ] 🟡 Получить согласие на бренд + домены (`loliscp.ru`, `api/cdn/socket/connect.loliscp.ru`)
- [ ] 🟢 Уточнить у основателя точную версию SCP: SL последнего прода (наша оценка — v14.x)
- [ ] 🟢 Через YouTube-канал прощупать живость комьюнити (анонс «думаем вернуть FYDNE»)
- [ ] 🟡 Закрыть роль C#-разработчика в команде (LabAPI/Qurre)

**Гейт фазы:** если дамп БД и бренд недоступны → цель меняется на «наследник», старая аудитория возвращается без прогресса.
