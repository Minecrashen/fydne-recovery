Рабочее пространство по воскрешению проекта **FYDNE** (SCP: Secret Laboratory).
Здесь хранится технический аудит, планы восстановления и заметки команды.

## Как синхронизировать между устройствами

Этот репозиторий синхронизируется через GitHub. На другом устройстве:

```bash
git clone https://github.com/<owner>/fydne-recovery.git
cd fydne-recovery
# редактируешь...
git add -A
git commit -m "что изменил"
git push
```

Перед началом работы на любом устройстве — сначала `git pull`, чтобы подтянуть свежие правки.
